# 逆战未来枪械应用

一个用于展示和管理逆战游戏枪械信息的Web应用，支持武器信息展示、搜索、复制以及管理员后台管理功能。

## 功能特点

### 主要功能
- 🎯 **武器展示**：以卡片形式展示武器信息，包括名称、皮肤、ID、功能等
- 🔍 **搜索功能**：支持按武器名称搜索
- 📋 **一键复制**：点击复制按钮快速复制武器信息
- 📱 **响应式设计**：适配移动端和桌面端
- 🎨 **现代UI**：采用玻璃态效果和渐变背景，视觉效果美观
- 🔄 **自动同步**：支持数据自动同步和多设备数据一致性

### 管理员功能
- 🔐 **管理员登录**：安全的管理员登录系统
- ➕ **添加武器**：支持添加新的武器信息
- ✏️ **编辑武器**：修改现有武器信息
- 🗑️ **删除武器**：删除不需要的武器
- 📤 **数据导出**：将武器数据导出为JSON文件
- 📥 **数据导入**：从JSON文件导入武器数据
- 🔄 **密码修改**：支持修改管理员密码

### 数据同步功能
- 📊 **数据同步工具**：专用的`sync.html`页面进行数据管理
- 📋 **嵌入代码生成**：生成可嵌入HTML的JavaScript代码
- 🔄 **自动同步检查**：定期检查数据变更并提示同步
- 📱 **多设备同步**：支持在不同设备间同步数据
- 💾 **自动备份**：自动创建数据备份，防止数据丢失
- 📈 **数据预览**：多种格式预览数据（表格、JSON、列表）

## 技术栈

- **HTML5** - 页面结构

- **JavaScript** - 交互功能
- **Tailwind CSS** - 样式框架
- **Font Awesome** - 图标库

## 如何使用

### 基本使用
1. 下载项目文件
2. 打开 `index.html` 文件即可使用
3. 浏览武器卡片，点击查看详情或复制信息

### 部署到服务器
1. 将所有文件上传到Web服务器
2. 通过浏览器访问对应URL

### 数据同步使用

#### 数据导出与同步
1. 访问 `sync.html` 页面
2. 配置导出选项（包含时间戳、压缩数据等）
3. 点击"导出数据文件"保存备份，或点击"生成嵌入代码"
4. 复制生成的JavaScript代码
5. 将代码粘贴到 `index.html` 的 `<head>` 标签内
6. 保存文件完成同步

#### 自动同步设置
1. 在 `sync.html` 页面中找到"自动同步设置"
2. 启用自动同步功能
3. 设置同步检查间隔（1分钟-1小时）
4. 设置变更阈值（达到指定变更数量时提示同步）
5. 点击"保存同步设置"

#### 多设备数据同步流程
1. 在设备A上更新数据（添加/编辑武器）
2. 系统提示"数据已更新，建议同步"
3. 生成嵌入代码并更新HTML文件
4. 在设备B上打开更新后的HTML文件
5. 数据将自动加载到设备B的本地存储

## 数据说明

### 武器数据结构
武器数据存储在浏览器的LocalStorage中，包括以下字段：
- `name` - 武器名称
- `id` - 武器ID
- `skin` - 皮肤名称（可选）
- `function` - 功能或用途
- `type` - 武器类型
- `copyText` - 复制文本
- `fullText` - 完整显示文本
- `description` - 武器描述
- `createdAt` - 创建时间（自动生成）
- `updatedAt` - 更新时间（自动生成）

### 同步数据结构
同步相关数据也存储在LocalStorage中：
- `syncSettings` - 同步配置（自动同步开关、检查间隔等）
- `backup_*` - 自动创建的数据备份（最多保留5个）

### 数据同步原理
1. **本地存储**：数据主要存储在浏览器LocalStorage中
2. **嵌入代码**：通过在HTML中嵌入JavaScript代码实现数据持久化
3. **哈希检测**：使用哈希算法检测数据变更
4. **自动检查**：定期检查数据状态，发现变更时提示同步

## 常见问题

### 数据同步问题
**Q: 数据同步后不生效怎么办？**
A: 尝试刷新页面或清除浏览器缓存后重新访问

**Q: 如何在多台电脑间同步数据？**
A: 使用数据导出功能，将数据文件复制到其他电脑后导入，或更新HTML文件中的嵌入代码

**Q: 同步提示频繁出现怎么办？**
A: 在同步设置中调整变更阈值或检查间隔

### 其他问题
**Q: 忘记管理员密码怎么办？**
A: 清除浏览器本地存储后使用默认密码登录

**Q: 数据丢失了怎么恢复？**
A: 检查是否有自动备份，或使用之前导出的数据文件恢复

**Q: 应用运行缓慢怎么办？**
A: 减少武器数据量，或清理浏览器缓存
